# 🌀 Fibergo Kickstarter


🤠 Cool Template for Fibergo includes JWT authentication and MongoDB connection.


# 🦾 Features
  
-[] using Fibergo

-[] explanations for function and instances

-[] MongoDB connection

-[] JWT authentication

-[] Perfect structure
